## Key Design Principles

1. **Emoji-First**: Use emojis for quick visual scanning
2. **Action-Oriented**: Every message has clear next steps
3. **Progressive Disclosure**: Show summary, link to details
4. **Friendly Tone**: Conversational, not robotic
5. **Mobile-Optimized**: Works great on phones
6. **Consistent Patterns**: Same layout for similar messages
7. **Real-time Updates**: Show progress, not just results
8. **Error Recovery**: Always show how to fix issues

---
